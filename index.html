Recent Projects

Optimal Sequential AlexNet(OSAN) for Tuberculosis classification

Developed breakthrough deep learning algorithm OSAN, achieving 99.6% accuracy in classifying TB from chest X-rays.

Bone fracture classification on Musculoskeletal(MURA) dataset

Developed a state-of-the-art bone fracture classification system on the MURA dataset - Stanford ML group. Customised the architecture of InceptionResNetV2 neural network by adding sequential layers.

Early Celebral disease detection

A Python implementation of a pose estimation model using BlazePose MediaPipe. The model takes input from the webcam or a video file, calculates angles of the body joints, and displays the stage as either "normal" or "effected" depending on the threshold angle. The model also displays the frames per second (FPS) of the video.

Gender Classification with YOLOv7

The project on custom object detection using YOLOv7 and gender classification aimed to develop a deep learning model that can detect and classify objects in images or videos. The model was trained using the YOLOv7 architecture and a dataset of images that were manually labeled for object detection.

Object Detection on CCTV Footage

Object detection with Darknet and YOLOv4. Pretrained weights were used to train the model and track objects.The model detects person, bag, tie , backpacks and handbags. The model can also be enhanced and customised by training it with various weights


Web Application

This web application displays all the relevant details of the queried movie. I used OMDB API inorder to fetch the details of all movies. Firebase was used inorder to validate the sign in credentials of the users. The application was built using html, css, javascript.

About me

Computer Science Undergraduate


Hello! I'm Mayukha Mandya Ammangatambu, a passionate and driven 4th-year undergraduate student at BVRIT Hyderabad College of Engineering for Women. As I near the completion of my undergraduate studies, my journey has been marked by a deep fascination with the intersection of technology, ethics, and artificial intelligence. Throughout my academic tenure, I have actively engaged with various solutions and architectures, tirelessly seeking ways to enhance their capabilities. However, one crucial aspect has consistently captured my attention: the lack of explainability in existing models. This realization has become the driving force behind my desire to pursue a master's degree, focusing on Ethical Deep Learning. My goal is to contribute to the field by researching and developing solutions that not only push the boundaries of artificial intelligence but also prioritize explainability. I believe that making AI understandable is paramount to its ethical deployment and acceptance in society.
