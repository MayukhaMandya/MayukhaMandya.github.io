Portfolio

Optimal Sequential AlexNet (OSAN) for Tuberculosis Classification

Developed a customized deep learning algorithm built on the traditional AlexNet. The neural network achieved an accuracy of 99.6% and a Cohen's Kappa score of 0.98 in classifying Tuberculosis-infected chest X-rays.

Bone Fracture Classification on Musculoskeletal (MURA) Dataset

Developed a bone fracture classification system on the MURA dataset - Stanford ML group. Customized the architecture of InceptionResNetV2 neural network by adding sequential layers.

Early Cerebral Disease Detection

A Python implementation of a pose estimation model using BlazePose MediaPipe. The model takes input from the webcam or a video file, calculates angles of the body joints, and displays the stage as either "normal" or "affected" depending on the threshold angle. The model also displays the frames per second (FPS) of the video.

Gender Classification with YOLOv7

The project on custom object detection using YOLOv7 and gender classification aimed to develop a deep learning model that can detect and classify objects in images or videos. The model was trained using the YOLOv7 architecture and a dataset of images that were manually labeled for object detection.

Object Detection on CCTV Footage

Object detection with Darknet and YOLOv4. Pretrained weights were used to train the model and track objects. The model detects person, bag, tie, backpacks, and handbags. The model can also be enhanced and customized by training it with various weights.
